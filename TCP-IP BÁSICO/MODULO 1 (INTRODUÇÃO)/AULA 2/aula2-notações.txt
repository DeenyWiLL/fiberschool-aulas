* o sistema binário é um sistema de dois digitos, representados por "0 e 1"

* os aparelhos eletrônicos (celulares, computadores, etc), e a informática no geral, se comunicam e interpretam através do sistema binário

* o sistema binário converte dados (como por exemplo, um número "25") para binário, onde o computador irá interpretar e entender os dados 

* a conversão, por exemplo, do número 25 funciona da seguinte forma:
25/2 = 12/2 = 6/2 = 3/2 = 1
você sempre irá dividir o número por 2, até restar o número 1 ou 0
as sobras, será seu resultado final
sendo 1|0|0|1|1, onde a leitura do codigo binario é da direita para esquerda, onde 25 em binário é igual a 11001

* existe uma tabela de conversão que funciona da seguinte forma:

caso você queira converter o número 25, você irá selecionando os numeros mais próximos de 25, até chegar no número desejado, por exemplo:

	2^9 | 2^8 | 2^7 | 2^6 | 2^5 | 2^4 | 2^3 | 2^2| 2^1 | 2^0
	512 | 256 | 128 | 64  | 32  | 16  |  8  |  4 |  2  |  1
	    |     |     |     |     |  .  |  .  |    |     |  .

os quadrados que ficaram sem nenhum espaço entre os números desejados, você vai preencher com 0, e os quadrados que ficaram marcados, você vai preencher com
o número 1

ficando = 11001

* é importante saber disso, pois:

- os dispositivos processam em bits
- os pacotes de dados que são transmitidos são bits
- as unidades de medida são bits 

* um bit é digito, um Byte é um conjunto de 8 digitos 

- 1 Byte = 8 bits
- 1 Kilobyte (KB) = 1024 bits
- 1 Megabyte (MB) = 1024 kilobytes
- 1 Gigabyte (GB) = 1024 megabytes

* em rede de telecomunicações, as velocidades são em bits

- uma internet de 100mbps (100 mega bits por segundo) equivale a 100 milhões de bits por segundo
- uma internet de 1Gbps (1 giga bit por segundo) equivale a 1 bilhão de bits por segundo
