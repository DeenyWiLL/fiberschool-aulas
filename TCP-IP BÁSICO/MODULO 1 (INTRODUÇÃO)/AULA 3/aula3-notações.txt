* Modelo OSI - Open Systems Interconnection
Criado pela International Organization for Standardization (ISO) em 1984
o modelo OSI foi criado com o objetivo de padronização de comunicação entre sistemas de computadores

* existem 7 camadas de referência do modelo OSI

7 - Aplicação: fornece serviços para apps que utilizam a rede, como e-mail, navegar na web, etc. permite que dispositivos se comuniquem entre si e define protocolos
e formatos de dados utilizados para essa comunicação

6 - Apresentação: é responsável pela formatação dos dados para que possam ser interpretados pelo receptor. ela é responsável pela criptografia

5 - Sessão: estabelece, gerencia e encerra conexão entre apps em dispositivos diferentes. ela fornece funções para controle de dialogo, para que as sessões
de comunicação entre apps possam ser estabelecidas e encerradas corretamente. 

4 - Transporte: fornece serviços de transferência de dados confiáveis entre apps em diferentes dispositivos. garante que a entrega de dados ocorra de forma correta
sem interrupções ou perca de fluxo.

3 - Rede: responsável pelo roteamento de dados de um dispositivo para o outro em uma rede. define padrões para criação de pacotes de dados, contendo dados como
endereço de origem e destino, e gerencia o fluxo de tráfego de rede. também fornece controle de congestionamento.

2 - Enlace: fornece um serviço confiável de transmissão de dados de um dispositivo para o outro em uma rede local. ela é responsável por dividir os dados recebidos
em frames e adicionar informações de controle para ajudar a detectar erros na transmissão. gerencia também o acesso ao meio fisíco, evitando que colisões de dados
entre dispositivos que compartilham o mesmo meio

1 - Física: é responsável pela transmissão de bits brutos de um dispositivo para outro através de um meio físico, como cabos, fibra óptica, etc. define padrões 
para transmissão de dados, como tensão elétrica, frequência, velocidade de transmissão e comprimento máximo do cabo.
